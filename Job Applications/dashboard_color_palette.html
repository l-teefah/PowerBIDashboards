Usage Guidelines

  • Background hierarchy: Use #1a1d29 for main background, #2a2f3a for cards, #363b4a for nested elements
  • Data visualization: Use blue scale (#4fc3f7 → #0d47a1) for primary data series
  • Status indicators: Green for success, Orange for pending, Red for rejection, Purple for interviews
  • Text contrast: Always ensure minimum 4.5:1 contrast ratio against background colors
  • Interactive elements: Use #4fc3f7 for hover states and active selections
  • Charts & graphs: Cycle through blue scale first, then introduce supporting colors
